## üìå Project Overview
This project aims to analyze urban expansion and land use changes using **Sentinel-1/2 (SAR/optical) and Landsat satellite data**. We focus on detecting and classifying newly artificialized areas (e.g., buildings, roads, parking lots) and comparing the performance of SAR vs. optical imagery for land use monitoring.

### Key Objectives:
- Detect artificialization using **SAR image comparison**.
- Compare results with **optical imagery** at the same locations (precision, computational cost).
- Classify new artificialized areas (e.g., buildings, roads, parking lots).
